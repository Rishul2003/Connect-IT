## Installation

git clone https://github.com/Rishul2003/Connect-IT.git

The  Project runs at Port 5173

  
  For Frontend :
```bash
  cd client
  npm Install
  npm run dev
```
For Backend:
```bash
  cd server
  npm Install
  npm start
```
For Socket:
```bash
  cd server
  npm Install
  npm start
```

    
## Tech Stack

**Client:** React, javascript, scss, material ui 

**Server:** Node, Express,  jwt , bcrypt, multer , cloudinary, socket.io

**Database:** MongoDB
